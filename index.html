KlipQ

A Clipboard Manager for macOS

Every great invention starts with a need. When I'm writing code, there are often times when I wish I could copy multiple things in a row, and paste them in a row. Times when I wish I could paste that item I copied four copies ago. Enter: KlipQ

KlipQ is a clipboard manager I made for the very simple purpose of saving a history of your recently copied items, and allowing you to paste directly from that history. My hope is that others get as much use out of this as I do.

Obsidian Plugin

Daily Notes Manager

Enhance and customize your daily notes

Daily Notes Manager

Customize your Obsidian Daily Notes

Obsidian is my must-have tool for notetaking. The Daily Notes feature makes it easy to create a unique note every day. I realized a feature I needed was to have one daily note to include content from other notes, so I created a Daily Notes Manager to help.

I use it to carry over unfinished tasks from the previous day, rollover a "yesterday's work" summary into my "Daily Standup" section, archive old notes, create custom organization based on day of the week, and more.

Firefox Theme

Gruvfox

Gruvbox theme with a private-browsing-only option

Gruvfox

A new kind of Firefox theme

Making a Firefox theme is easy. Making one that only applies to Private Browsing? Less so.

Gruvfox is a gruvbox theme for Firefox that directly modifies the CSS of the browser to create a unique type of theme. This theme can be applied in regular browsing, private browsing, or both.
